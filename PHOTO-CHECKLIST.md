# Nebula Forge — Photos We Need From You

Two sets of photos: **29 individual parts** (for the configurator) and **3 full builds** (for the homepage / product rigs).

**How to use them once you have them:**
- **Parts** → name each file EXACTLY as shown (e.g. `gpu1.png`), drop it in `assets/parts/`, then uncomment its line in `assets/parts/photos.js`.
- **Builds** → name each file as shown (e.g. `quasar.jpg`), drop it in `assets/parts/`, then uncomment its line in `assets/parts/rigs.js`.
- Missing photos fall back to a clean branded plate automatically — nothing breaks if some aren't ready yet.

**Shooting tips for a uniform, premium look:** plain white or black backdrop, knock out the background to transparent (save parts as `.png`), square-ish 1:1 framing, part centered, ~2000px. Keep the same angle + lighting within each category.
